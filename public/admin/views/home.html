Aplikasi Ujian Berbasis Web

Menggunakan back-end NodeJS dan Front-end sesuai keinginan klien. Karena back-end dibuat dengan menggunakan metode Restful API, sehingga front-end hanya perlu mengirim dan menerima data dari server dalam bentuk JSON (JavaScript Object Nation).
Fitur :
  • Fitur manajemen Ujian, Soal ujian dan peserta ujian
  • Fitur manajemen Soal
  • Fitur manajemen Peserta
